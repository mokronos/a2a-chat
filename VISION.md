# Vision

This should be a simple component library that provides:
- A simple chat interface (input, messages, etc.)
- A way to connect that component to an a2a server
    - The connection should be programmatically managed
    - It will be an issue with cors anyways
    - Could specify a backend url in the component and then auto change the entered (cors) url to the proxy url in the backend (if proxy exists)

- Maybe have the input for the url optional for the component

This should not be a full interface, just a component.
